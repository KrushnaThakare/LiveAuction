# Screenshot Shot List — User Manual Picture Edition

Use this list to capture screenshots for the user manual.  
**Recommended:** 1920×1080 browser window, dark theme (Midnight Blue), sample tournament with 3+ teams and 10+ players.

Log in as Super Admin or Operator before capturing admin screens.

---

## How to capture

1. Open the URL listed below.  
2. Use **Windows:** `Win + Shift + S` | **Mac:** `Cmd + Shift + 4` | **Chrome:** DevTools → ⋮ → Capture screenshot (full size).  
3. Save as `docs/screenshots/XX-description.png` using the filename in each row.  
4. Insert into your PDF/Word copy of `USER_MANUAL.md`.

## Tips for professional manual PDF

- Use the same tournament name in all shots (e.g. “Demo Premier League 2026”).  
- Blur or use fake phone numbers in registration/sold screenshots.  
- For overlay shots, use OBS preview or full-screen browser at 1920×1080.  
- Add red numbered callouts in PowerPoint/Figma pointing to buttons described in the manual.
